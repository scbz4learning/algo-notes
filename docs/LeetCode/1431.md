## Quesiton 

> **题目：拥有最多糖果的孩子**
>
> 有 `n` 个孩子和糖果。给定一个整数数组 `candies`，其中 `candies[i]` 表示第 `i` 个孩子拥有的糖果数量，并且给定一个整数 `extraCandies`，表示你拥有的额外糖果数量。
>
> 返回一个布尔数组 `result`，其中 `result[i]` 为 `true` 表示在给第 `i` 个孩子所有额外糖果后，他们将拥有最多的糖果，或者为 `false` 表示他们仍然没有最多的糖果。
>
> 注意：多个孩子可能拥有最多的糖果。
>
> **例子 1**：
> 输入: `candies = [2,3,5,1,3]`, `extraCandies = 3`  
> 输出: `[true,true,true,false,true]`
>
> **解释**：
> 如果给每个孩子所有额外糖果：
> - 孩子 1：2 + 3 = 5 个糖果，是最多的。
> - 孩子 2：3 + 3 = 6 个糖果，是最多的。
> - 孩子 3：5 + 3 = 8 个糖果，是最多的。
> - 孩子 4：1 + 3 = 4 个糖果，不是最多的。
> - 孩子 5：3 + 3 = 6 个糖果，是最多的。
>
> **例子 2**：
> 输入: `candies = [4,2,1,1,2]`, `extraCandies = 1`  
> 输出: `[true,false,false,false,false]`
>
> **解释**：
> 只有 1 个额外糖果：
> - 孩子 1 永远是最多糖果的，即使其他孩子得到了额外的糖果。
>
> **例子 3**：
> 输入: `candies = [12,1,12]`, `extraCandies = 10`  
> 输出: `[true,false,true]`
>
> **限制**：
> - `n == candies.length`
> - `2 <= n <= 100`


## Solution

```cpp
class Solution {
public:
    vector<bool> kidsWithCandies(vector<int>& candies, int extraCandies) {
        int maxCandies 
            = * std::max_element(
                    candies.begin(),
                    candies.end()
                );
        
        vector<bool> ans;
        for (auto c : candies) {
            if (c + extraCandies >= maxCandies)
                ans.push_back(true);
            else
                ans.push_back(false);
        }
        return ans;
    }
};
```
