## Question

> **题目：字符串的最大公约数**
>
> 给定两个字符串 `str1` 和 `str2`，我们称 `t` 可以“整除” `s`，当且仅当 `s = t + t + t + ... + t`（即 `t` 自身重复多次）。  
> 返回一个最大字符串 `x`，使得 `x` 可以同时整除 `str1` 和 `str2`。
>
> **例子 1**：
> 输入: `str1 = "ABCABC"`, `str2 = "ABC"`  
> 输出: `"ABC"`
>
> **例子 2**：
> 输入: `str1 = "ABABAB"`, `str2 = "ABAB"`  
> 输出: `"AB"`
>
> **例子 3**：
> 输入: `str1 = "LEET"`, `str2 = "CODE"`  
> 输出: `""`
>
> **例子 4**：
> 输入: `str1 = "AAAAAB"`, `str2 = "AAA"`  
> 输出: `""`
>
> **限制**：
> - 1 <= `str1.length`, `str2.length` <= 1000
> - `str1` 和 `str2` 由大写英文字母组成。

## Solution

```cpp
class Solution {
public:
    string gcdOfStrings(string str1, string str2) {
        if (str1 + str2 != str2 + str1)
            return "";
        
        int l1 = str1.size(), l2 = str2.size();
        
        while (l2 != 0) {
            auto temp = l1 % l2;
            l1 = l2;
            l2 = temp;
        }

        return str1.substr(0, l1);
    }
};
```

## 总结
1. 连续公共子串, 左右拼接两个字符串
2. gcd: 欧几里得算法
