## Question

> **题目：判断子序列**
>
> 给定两个字符串 `s` 和 `t`，如果 `s` 是 `t` 的子序列，则返回 `true`，否则返回 `false`。
>
> 字符串的子序列是从原始字符串中删除一些（可以不删除）字符后形成的新字符串，并且不改变剩余字符的相对位置。  
> 例如："ace" 是 "abcde" 的子序列，而 "aec" 不是。
>
> **例子 1**：
> 输入: `s = "abc"`, `t = "ahbgdc"`  
> 输出: `true`
>
> **例子 2**：
> 输入: `s = "axc"`, `t = "ahbgdc"`  
> 输出: `false`
>
> **限制**：
> - `0 <= s.length <= 100`
> - `0 <= t.length <= 10^4`
> - `s` 和 `t` 仅包含小写英文字母。

## Solution

要想明白 loop 中, 做单步操作, 不要 while 嵌套

```cpp
class Solution {
public:
    bool isSubsequence(string s, string t) {
        int i = 0, j = 0;
        while (j < t.size()) {
            if (s[i] == t[j])
                i++;
            j++;
        }
        return i == s.size();
    }
};
```
