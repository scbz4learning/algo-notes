## Question

> **题目：移动零元素**
>
> 给定一个整数数组 `nums`，将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
>
> 注意：你必须在原数组上进行操作，不可以使用额外的数组空间。
>
> **例子 1**：
> 输入: `nums = [0,1,0,3,12]`  
> 输出: `[1,3,12,0,0]`
>
> **例子 2**：
> 输入: `nums = [0]`  
> 输出: `[0]`
>
> **限制**：
> - `1 <= nums.length <= 10^4`
> - `-2^31 <= nums[i] <= 2^31 - 1`


## Solution
### Naive

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0, fast = 0;
        while (fast < nums.size()) {
            while (fast < nums.size() && nums[fast] == 0) {
                fast++;
            }
            if (fast < nums.size()) {  
                auto temp = nums[slow];
                nums[slow] = nums[fast];
                nums[fast] = temp;
                slow++;
                fast++;
            }
        }
    }
};
```

### 单步
要记得 loop 中 完全是可以单步的.

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0, fast = 0;
        while (fast < nums.size()) {
            if (nums[fast] == 0) {
                fast++;
            } else {
                swap(nums[fast], nums[slow]);
                fast++;
                slow++;
            }
        }
    }
};
```

或者更好地: 

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0, fast = 0;
        while (fast < nums.size()) {
            if (nums[fast] != 0) {
                swap(nums[fast], nums[slow]);
                slow++;
            }
            fast++;
        }
    }
};
```
