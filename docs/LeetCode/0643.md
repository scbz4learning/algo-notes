## Question

> **题目：最大平均子数组 I**
>
> 给定一个整数数组 `nums`，长度为 `n`，以及一个整数 `k`。
>
> 找出一个长度为 `k` 的连续子数组，计算其平均值，并返回该值。任何计算误差小于 `10^-5` 的答案都可以接受。
>
> **例子 1**：
> 输入: `nums = [1,12,-5,-6,50,3]`, `k = 4`  
> 输出: `12.75000`
>
> **解释**：
> 最大平均值为 `(12 - 5 - 6 + 50) / 4 = 51 / 4 = 12.75`。
>
> **例子 2**：
> 输入: `nums = [5]`, `k = 1`  
> 输出: `5.00000`
>
> **限制**：
> - `n == nums.length`
> - `1 <= k <= n <= 10^5`
> - `-10^4 <= nums[i] <= 10^4`



```cpp
class Solution {
public:
    double findMaxAverage(vector<int>& nums, int k) {
        double curSum = 0;
        for (int i = 0; i < k && i < nums.size(); i++) {
            curSum += nums[i];
        }
        double maxSum = curSum;
        for (int i = k; i < nums.size(); i++) {
            curSum = curSum - nums[i-k] + nums[i];
            maxSum = max(maxSum, curSum);
        }
        return maxSum / k;
    }
};
```
